# Managing Available Resources

- Show overall disk usage
```bash
df -h
```

- Show disk usage for a particular directory
```bash
du -sh /bin/
```

- Show available memory
```bash
free -h
```

- Show cpu uptime
```bash
uptime
```

- Show cpu details
```bash
lscpu
```

- Show pci device details
```bash
lspci
```

- Show usb devices
```bash
lsusb
```
